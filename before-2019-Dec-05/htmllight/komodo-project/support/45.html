If you are using the Lite wallet (recommended), when you first load Agama it will ask you for a seed, or it can create one for you. That's your Key. Keep it safe, its the only way to get to your funds. Copy the address, log out, then log back in to make sure you see the exact same address, then you know you retyped (or pasted) the correct seed. You should send 1 kmd to the wallet first to make sure you can access it again. If you see the kmd you sent, then you should be good. (edited)
