Steps to test zmigrate 1. Stop the chain and build latest
cd ~/komodo/src ./komodo-cli -ac_name=RKT stop git pull make
2. Compile dapp
./cc/dapps/makedapps
3. Start the chain (don't start mining)
./komodod -ac_name=RKT -ac_supply=0 -ac_reward=25600000000 -ac_halving=77777 -ac_private=1 -addnode=54.36.126.42 -addnode=94.130.224.11 &
4. Create a sprout address and shield mined funds there
./komodo-cli -ac_name=RKT z_getnewaddress sprout ./komodo-cli -ac_name=RKT z_shieldcoinbase "*" zcdiaH6hVy8vuTiCwerBoGPt2fpthQo6KEPDZJwiMekqsj3mhXYLiGZ4SzvbyVZzQbqAp94fH761imGfHa4cc3tUqnuaDo7 # change to your zc address
5. Create and sapling z addr and initiate migrate
./komodo-cli -ac_name=RKT z_getnewaddress ./zmigrate RKT zs1fp7h7y348l6nm4qwwqq2ntu8vjtsgkmf7ghntsd72tef2wkzdh8ppyeegv9klaj4zzcpupfr0kd # change to your sapling z address
6. Wait for migrate to be done. It may take a while at the end, don't worry and don't cancel. 7. Use z_listunspent call and search for and address that starts with zc. If none, that means the migrate has been successful for you.
./komodo-cli -ac_name=RKT z_listunspent
