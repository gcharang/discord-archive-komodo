How about that 5% annual staking reward, how does that work then? @CoinKing88 Part of the problem with you understanding how it works is not understanding the terminology behind it and the terminology you are using. There is no staking in KMD. KMD does have a "usage reward" where you get a bonus reward for using it. Merely sending a transaction that meets certain rules allows you to earn this reward. Staking requires you to actively produce a block. These are completely different things. The user rewards don't "come from" x or y, they are created with the transaction at the time of the transaction. The amount that users get as a reward does not exist before the user gets the reward. The rewards are created by the protocol as a form of emission. (edited)
