[08-May-19 09:55 PM] SHossain#0007 (pinned)
**Mine KMD or other Equihash coins using Nvidia GPU**
```
sudo apt-get update && sudo apt-get upgrade -y
sudo apt-get install build-essential automake -y
sudo apt-get install libcurl4-openssl-dev libssl-dev nvidia-cuda-dev nvidia-cuda-toolkit -y
git clone https://github.com/tpruvot/ccminer
cd ccminer
./build.sh

./ccminer --algo=equihash --url=stratum+tcp://127.0.0.1:3059 --user=RXmtP1PVH1ChWkL6EwM1p2LqJx8kMMwj88 --pass=x 
```
Change `127.0.0.1:3059` with pool URL and port and `RXmtP1PVH1ChWkL6EwM1p2LqJx8kMMwj88` with your own KMD address.
