dimxy 29-Jun-19 11:58 AM
The 'global address' is a feature common for every komodo cc contract. This address might be used for various purposes. It is like a shared account. Although anyone might spend funds from this address this spending is controlled by the validation code. Currently in marmara this address is used for storage of markers, that is, small amounts on this address which allow to enumerate the transactions which have such an utxo. By this we can find all 'marmaraissue' transactions for open credit loops. The marmara validation code will allow to spend such markers for the settlement transactions and by this close the credit loop. And yes we can store marmara's backup fund on this adddress too.
