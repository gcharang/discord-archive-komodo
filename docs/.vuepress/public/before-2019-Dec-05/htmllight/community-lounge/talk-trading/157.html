@Mische so im going to answer that with a caveat. I am familiar with kmd-cli and DNS-setup. It took me about 4 hours. Most of that time was to sync up the kmd chain (i did this on a new server). There are tutorials to set up a DNS, tutorials to install komodo, and tutorials to create the asset chain. You do need two instances of komodo running to do this.
Once you do it once, creating a new asset chain takes about 15 to 30 mins. Most of that time is deciding what parameters you want to create the chain with.
