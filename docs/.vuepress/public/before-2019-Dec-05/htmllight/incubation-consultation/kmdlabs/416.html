@Alright https://dash.plot.ly/authentication dash have nice (it looks simple to use at least)auth module At first install this dep:
pip3 install dash-auth==1.2.0 import dash_auth
Then add to code:
VALID_USERNAME_PASSWORD_PAIRS = [ ['parsed_username_from_config_file', 'parsed_password_from_config_file'] ]
Then add before layout:
auth = dash_auth.BasicAuth( app, VALID_USERNAME_PASSWORD_PAIRS )
 (edited)
