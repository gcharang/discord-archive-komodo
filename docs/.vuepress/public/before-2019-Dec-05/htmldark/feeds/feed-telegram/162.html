[telegram] <Bushybro> My advice to anyone have issues with their wallet, first and foremost please verify you are using the latest wallet version. You can do this by deleting the first letter and putting it back in and do this also with the very last letter in the seed. Third, know which mode you are in. Either it's native or spv. Native mode dowloads the whole blockchain and you have to backup the wallet.dat file. Spv mode only requires that you enter the same seed each time which brings you to the same address every time.
