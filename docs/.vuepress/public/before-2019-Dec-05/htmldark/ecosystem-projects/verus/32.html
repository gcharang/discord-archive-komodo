[dscrdvrsc] <miketout> @TexWiller made a FAQ some time ago, which is in the #faq-chosen channel, along with a lot of great stuff. From the FAQ: Q: How to consolidate multiple wallet.dat files in one? Answer: 1) Edit the VRSC.conf file on each instance you have, adding the following line: exportdir=/LOCAL_PATH/ (ie /home/user/) 2) Restart verusd 3) Issue the following command (on the verus-cli): z_exportwallet FILENAME (ie z_exportwallet export_instance01) 4) Copy the generated file to the machine who host your main wallet 5) Issue the following command (on the "main" verus-cli): z_importwallet /LOCAL_PATH/FILENAME (ie /home/user/export_instance01)
[dscrdvrsc] <miketout> you can do this on the GUI as well, but use the Settings->CLI to enter the z_exportwallet and z_importwallet commands make sure that before you do this, you backup all of your wallet files carefully

[dscrdvrsc] <miketout> you enter them as command line commands in the Settings->CLI </> section of the settings menu in the wallet. you need to select VRSC (native), then enter the command below and press the execute button.
